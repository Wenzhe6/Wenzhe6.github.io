Javascript概述

是由网景公司开发的一种跨平台,面向对象是网页脚本语言,是前端开发的核心语言,基于事件和驱动的解释性,松散型语言

js是一门什么样的语言?

  1. 松散型(是弱类型)
    · 强类型(对数据类型进行了严格的规划):int num 10;string str=’xx’; float f = 3.14
    · 但JS是var num = 10 var str=’xx’解析的时候会自动转换成对应的类型,定义变量时,不需要强类型,直接弱类型,变量的定义不需要声明类型
  2. 脚本语言
    · 其实JS不是一门真正的编程语言 真正的编程语言需要有编译的过程
    · 编译:把程序员写的代码转换成计算机能识别的语言(.class文件)
    · 脚本语言其实就是不需要编译的过程,编码后打开浏览器可以直接运行,由JS引擎来替我们解析执行
  3. 单线程
    · 在我们不主动开启新线程的情况下 默认js只有一个线程解析执行代码(还有三个辅助线程、但是它们都不参与解析的过程)
    如果中间遇到了阻塞 那么js代码就会停止执行 => 单线程的弊端 => js任务队列 => 事件轮询机制
    【任务队列】: 解决单线程的弊端
    js在执行过程中遇到了阻塞会停止执行,于是有了任务队列来解决这个问题
    【事件轮询机制】: 就是一个判断,不断判断
    任务队列是否还有任务,如果有,会将异步任务的回调函数放到任务队列中,然后继续执行同步任务

 遇到异步代码时,会将异步代码回调放到任务队列里,当zhi执行完代码后,再反过来执行任务队列里的代码
· 什么时候取出任务队列中的异步任务执行?
    需满足两个条件
        1. 主线程执行完毕 处于空闲状态
        2. 满足异步任务本身的触发条件
            a.定时器 (设定的时间到了)
            b.普通事件 onclick(点击、鼠标移入)

            c.http网络请求(数据有响应)
  1. 基于对象
    js并不是一门面向对象的语言,只是模拟了面向对象的思想
    面向对象:封装、继承(通过原型来模拟的)、多态(一个事物在不同情况下的表现形式 比如:水)
  2. 动态语言
    只有当代码真正执行到这一行的时候才能确定声明的变量的具体值
    num = 10
  3. 解释型语言
    · 由JS引擎来解析
    · 边执行边解释,在浏览器环境下运行

        · trim() 方法会从一个字符串的两端删除空白字符
            返回值: 一个代表调用字符串两端去掉空白的新字符串。

        · 字符串位置方法
            indexOf()
                功能：返回子字符串第一次出现的位置,如果没有找到该子字符串，则返回 -1 
                语法格式：string.indexOf('子字符串',下标) 下标是可选的
            slice()
                功能：可提取字符串的某个部分，并以新的字符串返回被提取的部分
                语法格式：str.slice(start，end)  
                注意:[start,end) 也就是说返回从start到end-1的字符
                包左不包右 => [start, end)
                返回值：一个新字符串
            substr()
                功能：可在字符串中抽取从开始下标开始到指定数目的字符
                语法格式：str.substr(start, length)
                注意：从开始下标截取长度位
                返回值：一个新字符串
            substring()
                功能：用于提取字符串中,介于两个指定下标之间的字符
                语法格式：str.substring(start, end) 
                返回值：一个新字符串
                start开始下标，和end结束下标，两个参数都为正数,下标如果是负值一律作为 0 处理
                返回从start到end-1的字符 包左不包右
                如果 start 比 end 大，那么该方法在提取子串之前会先交换这两个参数。
            charAt(index)
                功能：从一个字符串中返回指定的字符。
                一个介于0 和字符串长度减1之间的整数。 (0~length-1)
                如果没有提供索引，charAt() 将使用0。
            split() 
                功能:使用指定的分隔符字符串将一个String对象分割成子字符串数组，以一个指定的分割字串来决定每个拆分的位置。
                如果省略分隔符，被返回的数组, index [0] 中的包含整个字符串。
                    例如：str=”a/b/c” str.split()的结果是[“a/b/c”]
                如果分隔符是 ""，被返回的数组将是间隔单个字符的数组 。
                    例如：str=”a/b” str.split(“”)的结果是[“a”,”/”,”b”]
                如果分隔符是”/”，则按照分隔符去分割。
                    例如：str=”a/b”str.split(‘/’)的结果是 [“a”,”b”]
            replace()
                功能：在字符串中用一些字符替换另一些字符
                语法格式：str.replace(searchValue，replaceValue)
                返回值： 字符串

parseInt() && parseFloat() && toFixed()

· ① parseInt可以把任意值转换成数值 但是没有Number那么严格
  ② 如果第一个字符是数字会解析直到遇到非数字才会停止解析
  ③ 如果第一个是非数字字符 直接返回NaN
  ④ 一般用它进行取整 parseInt('12.34') => 12
    例: parseInt('1a') => 1  parseInt('a1') => NaN
· parseFloat() 把字符串转成浮点数
· parseFloat()和parseInt区别
    parseFloat会解析第一个. 遇到第二个.或者非数字结束,若解析的内容里只有整数,解析成整数
        例:
            parseFloat('12.34') => 12.34
            parseFloat('12.34.56') => 12.34
· toFixed(参数):小数点保留几位的方法
    参数: 保留几位 => 四舍五入
    应用场景: 购物车金额计算

运算规则

· 加法规则
    1.对于数值,不会产生任何影响
    2.只要有一个NaN,就为NaN
    3.正无穷和负无穷相加等于NaN
    4.只有字符串就不是加法,而是字符串的拼接
· 减法规则
    1.对于数值,不会产生任何影响
    2.只要有一个NaN就为NaN
    3.布尔值转换为对应的数值
    4.null "" 转成了 0
    5.非数值字符串转为NaN
· 除法规则
    1.对于数值,不会产生任何影响
    2.只要有一个NaN就为NaN
    3.布尔值转换为对应的数值
    4.null "" 转成了 0
    5.非数值字符串转为NaN
    6.对于无穷大的数值为Infinity
· 取余
    1.对于数值,不会产生任何影响
    2.只要有一个NaN就为 NaN
    3.对于无穷大的数值为NaN 
    4.布尔值转换为对应的数值 
    5.非数值字符串转为NaN
· 运算符优先级
    1.()优先级最高
    2. 一元运算符  ++   --   !
    3.算数运算符  先*  /  %   后 +   -
    4. 关系运算符  >   >=   <   <=
    5. 相等运算符  ==  !=   ===  !==
    6. 逻辑运算符  先&&   后||
    7. 赋值运算符  = 
· 逻辑运算符
    && 与
        1、遇见false则为false
        2、有一个操作数是 null,则返回 null
        3、有一个操作数是 undefined,则返回undefined
        4、有一个数是NaN,则返回 NaN
        [!Result]
            逻辑与运算符属于短路操作,顾名思义,如果第一个操作数返回是false,第二个数不管是true还是false都返回false
    || 或
        1、遇见true则为true
        2、有一个操作数是 null,则返回 null
        3、有一个操作数是 undefined,则返回undefined
        4、有一个数是NaN,则返回 NaN
    ! 非
        1、逻辑非运算符可以用于任何值。
        2、无论这个值是什么数据类型,这个运算符都会返回一个布尔值。
        3、它的流程是:先将这个值转换成布尔值,然后取反,规则如下
            !!"111" => Boolean("111") => true => !true => false => !false => true

自增自减

· ++在前 先自增再赋值
    ++在后 先赋值再自增
· 在没有赋值操作,前置和后置是一样的。
· 在赋值操作时:
    如果自增或自减运算符前置,那么前置的运算符会先累加或累减再赋值
    如果是后置运算符则先赋值再累加或累减。

基本数据类型和引用数据类型区别

  • 基本数据类型的赋值操作
    相当于将该基本数据类型拷贝了一份 拷贝后的数据和拷贝前的数据之间没有任何关系 彼此修改互不影响
    例:var a=3 var b=a a=5 clg(b)//3
  • 引用数据类型
    相当于将该引用数据类型的地址赋值给了另外一个变量 就相当于对该数据类型创建了一个”快捷键”,拷贝后的数据拷贝前的数据指向的是同一个地址值,所以拷贝前后的数据操作时同一块空间 修改数据会相互影响
    例:var obj={name:’zs’} var obj2=obj
      obj.name = 'lisi' clg(obj2.name) // 'lisi'

数组

  • 定义: 数组是【有序元素序列的集合】
  • 数组也是对象 type of [] == object
  • 数组的创建:
    1、创建时指定长度: var arr = new Array(5)
    2、创建时指定元素: var arr = new Array(1,2,3)
    3、字面量的创建方式: var arr = []
  • 数组元素的查找: 通过数组[下标] 下标是从0开始计算
  • 数组元素的修改: 通过数组[下标]找到该元素重新赋值
    例如:arr[1] = 66
  • 数组元素的清空: arr.length = 0 、 arr.splice(0)
  • 数组长度的获取: arr.length
  • 数组的遍历: for(var i=0;i<arr.length;i++){arr[i]}
  • 数组的方法:
    1. push(需要添加的元素) 方法将一个或多个元素添加到数组的末尾，并返回该数组的新长度(该方法修改原有数组)。
    2. unshift() 方法将一个或多个元素添加到数组的开头，并返回该数组的新长度(该方法修改原有数组)。
    3. pop()方法从数组中删除最后一个元素，并返回该元素的值。此方法更改数组的长度。
    4. shift() 方法从数组中删除第一个元素，并返回该元素的值。此方法更改数组的长度。
    5. indexOf(参数1, 参数2):方法返回在数组中可以找到一个给定元素的第一个索引，如果不存在，则返回-1。查找是全等 === 校验 需要校验类型
      参数1: 表示的是要查找的元素
      参数2: 表示的是开始查找的位置
    6. reverse() 方法将数组中元素的位置颠倒，并返回该数组。数组的第一个元素会变成最后一个，数组的最后一个元素变成第一个。该方法会改变原数组。
    7. concat(参数1) 方法用于合并两个或多个数组。此方法不会更改现有数组，而是返回一个新数组。
    8. sort(参数1) 方法用原地算法对数组的元素进行排序，并返回数组。默认排序顺序是在将元素转换为字符串，然后比较它们的UTF-16代码单元值序列时构建的
      参数1: [compareFunction] => 比较的函数 => 排序规则 => 如果省略 则按照unicode转换后的结果进行比较
       arr.sort(function(a,b){
     // 由小到大
     return a - b
     // 由大到小
     return b - a
 })
    9. join() 方法将一个数组（或一个类数组对象）的所有元素连接成一个字符串并返回这个字符串。如果数组只有一个项目，那么将返回该项目而不使用分隔符。
    10. splice(参数1,参数2,参数3) 方法通过删除或替换现有元素或者原地添加新的元素来修改数组,并以数组形式返回被修改的内容。此方法会改变原数组
      参数1: 从哪里开始删
      参数2: 删几个
      参数3: 要插入的项 (前面插入)
      返回值: 由被删除的元素组成的一个数组。如果只删除了一个元素，则返回只包含一个元素的数组。如果没有删除元素，则返回空数组。
    11. slice():方法返回一个新的数组对象，这一对象是一个由 begin 和 end 决定的原数组的【浅拷贝】（包括 begin，不包括end）。原始数组不会被改变。
    12. Array.from(): 方法对一个类似数组或可迭代对象创建一个新的，浅拷贝的数组实例,可以将伪数组转成真数组
      静态方法: 通过对象直接调用的方法我们称之为静态方法(原型)

面向对象

【Interview】 
    面试时问道面向对象思想可以参照以下思路回答:面向对象是一种开发思想 它将复杂的过程简单化 让我们从开发者变成了指挥者 指挥对象做事情 在实际开发过程中 我们需要面对成千上万的方法 而方法与方法之间又没有联系 于是使用面向对象的思想对方法进行了对象的封装 我们只需要面对几百个对象了 当我们需要使用到某个方法的时候 就去查找相关对象 我们认为一旦找到了该对象即可调用对象相关的方法
  • 面向对象代表 => java => 封装、继承、多态(一个事物在不同环境下的不同表现形式)、而js是基于对象、模拟面向对象的思想
  • JavaScript中的对象其实是生活中对象的一个抽象
    1. 事物的特征在对象中用属性来表示
    2. 事物的行为在对象中用方法来表示
    3. JavaScript的对象是【无序属性的集合】
      • 其属性可以包含【基本值】、【对象】或【函数】。对象就是一组没有顺序的值
      • 我们可以把JavaScript中的对象思想成【键值对 - 键:值 通过对象.键可以获取到值】,其中值可以是数据(任意类型)和函数
        例:var obj = {
        name:’zs’,
        age:18,
        loves:[‘playbasketball’,’’],
        girlFriend:{
          name:'王麻子'
        },
        job:true
        money: null
        }
  • 创建对象的方式
    1. 通过内置API提供的Object构造函数(可以认为是专门用来创建对象的函数)来实现 配合关键字 new 来创建实例对象
      例: var obj = new Object()
      new 会在堆内存中开辟了一块空间 并且返回了该空间的地址值(十六进制)我们可以通过地址值找到该对象从而调用对象的属性和方法
    • 不成文的规定
      如果函数名以大写字母开头 我们认为它是一个构造函数配合new关键字使用
如果函数名是小写字母开头 我们认为它是一个普通函数直接调用即可
      1. 使用字面量的方式创建对象 {}是对象的字面量 不是块级作用域
        var obj = {}
      2. 工厂函数 内部调用的也是new Object() || {} 生成一个对象并返回

BTW—工厂函数

  • 所谓工厂函数,就是指这些内建函数都是类对象,当你调用他们时,实际上是创建了一个”类实例”

作用域链

  • 只有函数可以制造作用域结构,那么只要是代码,就至少有一个作用域。凡是代码中有函数,那么这个函数就构成另一个作用域。
    如果函数中还有函数,那么在这个作用域中就又可以诞生一个作用域。将这样的所有的作用域列出来,可以有一个结构:函数内指向函数外的链式结构。就称作作用域链。
  • 作用域链查找规则(遵循就近原则)
    如果自己作用域中存在该变量或者函数,就直接使用该变量或者函数,如果本身作用域中没有,则往上级作用域中查找,直到找到全局作用域为止
  • 如果全局作用域中不存在该变量 则报错:xx is not defined
  • 如果全局作用域中不存在该方法 则报错:xx is not a function

立即行函数 && 回调函数

立即执行函数

  • 语法
    1. (function(形参){语句块})(实参)
    2. (function(形参){语句块}(实参))
  • 页面加载会立即执行 执行完毕立马从内存中消失 函数内的变量也会随之消失
  • 优缺点:
    优点是不会产生任何全局变量。
    缺点是无法重复执行。
  • 全局变量会在关闭当前页面后才从内存中销毁 所以不要定义太多的全局变量 否则会导致内存泄漏
  • 如果需要将自执行函数中的变量或方法暴露给外面使用 可以将window作为参数传递进来 然后将属性和方法挂载到window身上

回调函数(高阶函数)

  • 函数作为参数
    因为函数也是一种类型,可以把函数作为另一个函数的参数,在两个函数中调用
  • 函数作为返回值
    因为函数是一种类型,所以可以把函数可以作为返回值从函数内部返回

this指向

函数的上下文(this关键字)由调用函数的方式决定

函数如果不调用 则不能确定函数的上下文

圆括号直接调用函数，则函数的上下文是window对象

数组(类数组对象)枚举出函数进行调用,上下文是这个数组(类数组对象)

  • 数组下标
  • 伪数组 | 类数组 {0:’A’, 2:’B’, 3:’C’, length: 3}
  • arguments对象 document.getElementByTagName()
    特点:有类似数组解构 但不能调用数组的方法,如果需要调用:
    1. 将其转成伪数组Array.from(伪数组)

    2. [].prototype.map.call(伪数组)

IIFE(自执行函数)中的函数,上下文是window对象

  • (function(){})()

定时器、演示器调用函数,上下文是window对象

  • setInterval(函数，时间);
    setTimeout(函数，时间);

事件处理函数的上下文是绑定事件的DOM元素

  • DOM元素.onclick = function () {};

new操作符调用函数

  • new函数()
    JS规定,使用new操作符调用函数会进行”四步走”
    1. 函数体内会自动创建出一个空白对象{}
    2. 函数的上下文(this）会指向这个对象 this => {}
    3. 函数体内的语句会执行 {name:zs}
    4. 函数会自动返回上下文对象，即使函数没有return语句 return this

this关键字两种场景

  1. 构造函数中使用代表的是当前实例化对象 谁new了我 我就指向谁
  2. 全局环境中 this指向window 如果我们在全局中调用了函数 那么函数中的this就是指向window 谁调用我 我就指向谁

call、apply && bind

call

  • 作用: call()方法使用一个指定的this值和单独的给出的一个或多个参数来调用一个函数。

apply

  • apply与call大同小异 只是在参数处理有区别

bind

  • bind() 方法创建一个新的函数，在 bind() 被调用时，这个新函数的 this 被指定为 bind() 的第一个参数，而其余参数将作为新函数的参数，供调用时使用。

call、apply && bind 区别

  • 它们在功能上是没有区别的，都是改变this的指向，它们的区别主要是在于方法的实现形式和参数传递上的不同
    ① 函数.call(对象,arg1,arg2….)
    ② 函数.apply(对象，[arg1,arg2,…])
    ③ 函数.bind(对象,arg1,arg2,….)() 注：bind返回的是一个函数体，并不会直接执行函数

  • $(function(){}) 等价于 $(document).ready(function(){})
